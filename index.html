CS184 final project proposal

Peijie Li, Sharon Li

3D fluids simulations

In this final project, we would extend our interest in animination simulation and explore more about this interesting topics. From the lecture, we learned some about partical simulation which means that decomposes one object into smaller particals to simulate the movement for the shapeless object --- fluid, in this case. This is similar to what we have about cloth simulation. We decompose the cloth fiber into huge numbers of point mass and combine forces and strings connecting point mass to simulate the contraction between each points. However, this shoule be more interesting to apply particals to shape the fluids, because the fluids is not fixed shape and has strict restrications like the cloth simulation. Therefore, we want to use partical simulation to mimic the movement of the shapeless fluids.
