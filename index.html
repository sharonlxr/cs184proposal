3D fluids simulations and Snow Simulation

In this final project, we would extend our interest in animination simulation and explore more about this interesting topic. From the lecture, we learned some about partical simulation and grid-based simulation.

A little bit about contexts related to class

This is similar to what we have about cloth simulation. We decompose the cloth fiber into huge numbers of point mass and combine forces and strings connecting point mass to simulate the contraction between each points. However, partical-based means that decomposes one object into smaller particals to simulate the movement for the shapeless object --- fluid, in this case via position-based simulation. This shoule be more interesting to apply particals to shape the fluids and use position-based technics to manipulate the particals, because the fluids is not fixed shape and does not have strict restrications like the string in the cloth simulation. Therefore, we want to use partical simulation to mimic the movement of the shapeless fluids through predicting the positions for each points. In addition to fluid, we want to simulate snow by applying both partical and grid system. This should give us better understanding about both and compare the behaviours.

Goal

We would like to build a interactive system to play around with both simulation. For the fluids simulation, we can provide interactive system to added external force like winds or motion applied on the gui to control the direction of movement we are expecting. In the snow simulation, we want to create different scenes containing different objects and environments and we can use gui to shoot snow balls towards the scenes with the user applies motion to control direction via the gui.

In the fluid simulation, we want to first achieve wave movement and then fluid falling down. With the fluid falling down, we want to create a simulation of raining within a scenes containig some objects and enviroment to show the collision with the scenes and the different absorbing/perserving of rains of on the groud of the enviroment.
